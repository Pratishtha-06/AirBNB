# 🏡 **Airbnb Clone**

A full-stack Airbnb clone built using the MERN stack, allowing users to explore listings, make bookings, and manage their stays with a seamless experience.

# 🚀 **Features**

🔐 JWT-based Authentication (Login/Signup)

🏠 Create, view, and manage property listings

💾 Save/Favorite listings for later

💳 Secure payment integration

📅 Booking functionality with date selection

📱 Responsive and user-friendly UI

🗺️ Map Integration


# 🛠️ **Tech Stack**

-Frontend: React, CSS

-Backend: Node.js, Express.js

-Database: MongoDB

-Authentication: JWT

-Payments: Razorpay


# 📂 **Project Setup**

1. Clone the repository
   
   ```
   git clone https://github.com/your-username/airbnb-clone.git
   cd airbnb-clone
   ```

3. Install dependencies
   
   ```
   npm install
   cd client
   npm install
   ```

5. Run the project

   ```
   #Run backend
   npm start

   #Run frontend 
   cd client 
   npm start
   ```

# **🔑 Environment Variables**
   ```
   MONGO_URI
   JWT_SECRET
   PAYMENT_API_KEY
   ```
